Seed Ratings

One-shot replay of all historical games to seed Glicko-2 ratings under tally_kotb_pickup/ratings.

What this does

Reads every result from:

  • tally_kotb/kings/results (current Kings season)
  • tally_kotb/queens/results (current Queens season)
  • tally_kotb/archive/*/{kings,queens}/results (all archived seasons)
  • tally_kotb_pickup/archive/*/results (any closed pickup events)

Sorts every game chronologically by timestamp, replays each through the Glicko-2 module with the same per-player updates the live apps use, and writes final state to tally_kotb_pickup/ratings/.

Heads up: this overwrites any existing ratings under tally_kotb_pickup/ratings/. Only run it once, before you turn on live rating updates.
